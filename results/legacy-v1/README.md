# Frozen v1 observations

These schema-4 reports are retained for auditability. They used forced-choice vignette suffixes,
flat string aliases, a binary `MustAbstain` label, and negative-only counterfactuals. Clinical review
found that several ablations supported probable diagnoses and several negative tests did not exclude
the original diagnosis. The old scores therefore must not be quoted as current benchmark findings.

Do not rescore or rewrite these files under the v2 contract; use the schema-5 artifacts one directory
above for current results.
